# Changelog

## [0.1.0] - 2026-03-05
- Initial release of Ressa Noise Detector App.
- Added Electron + Vite + React + TypeScript desktop app shell.
- Added live microphone RMS noise measurement with 0-100 normalized level.
- Added scrolling canvas chart, live meter, and Quiet/Normal/Loud status labels.
- Added Ressa face states (calm, annoyed, angry) with loud shake animation.
- Added controls for start/stop, sensitivity, thresholds, and test mode.
- Added privacy-focused docs: README, PRIVACY, and ARCHITECTURE.
